# End-to-end-NLP-Project-Implementation-For-Hate-Speech-Classification

## How to run?

```bash
conda create -n hate python=3.8 -y
```

```bash
conda activate hate
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```
